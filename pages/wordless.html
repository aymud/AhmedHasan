Human-Computer Interaction

Project Name: Wordless

Visual Summary

Project Description

A mobile web browser game created using web sensors in order to provide a rich interaction with non-verbal audio input.



Interaction Design

Playing around with the different sensors and phone constraints to create an interactive game. This led me to design it in such a way so that the user feels engaged and that they have full control.



Design Rationale

To add a bit more oomph to the game, I added bombs which would need to be avoided. To take advantage of the sensors and put a more expressive spin on the voice input, I added larger moles which would require the player to make louder noises to whack them.
